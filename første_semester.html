Before and after In this exercise we had to go out and take a picture to photoshop with a few goals and criterias to fufill:

  • Remove an element while also filling out the remove area
  • Add an element but make it seamless so it looks like it belongs
  • Adjust the hue and saturation or add a filter
  • Crop and adjust tje horizontal line if the image is crooked
  • Format the image aspect ratio to be 4:3 or 3:4 and a image size of 1 M pixel or less

    • Give the button a pres and see if you can spot all the differences!
